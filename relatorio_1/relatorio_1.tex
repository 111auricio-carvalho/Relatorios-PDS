\chapter{Introdução}
Este relatório tem como objetivo apresentar os resultados obtidos no primeiro laboratório da disciplina de Processamento Digital de Sinais (PDS), que aborda o tema de Sinais Fundamentais Discretos.

Nesta prática, explorou-se a geração e plotagem de sinais discretos no tempo, bem como operações matemáticas entre eles, através de rotinas desenvolvidas na ferramenta Octave.

\chapter{Metodologia e Procedimentos Práticos}

De acordo com o roteiro do laboratório, foram realizadas as seguintes atividades no Octave:

\section{Parte I - Sinais Fundamentais Discretos}
Foi solicitado gerar e plotar os seguintes sinais fundamentais discretos no tempo, na faixa de $-20 \leq n \leq 40$:
\begin{itemize}
    \item \textbf{a)} Função impulso unitário.
    \item \textbf{b)} Função degrau unitário.
    \item \textbf{b)} Função rampa.
    \item \textbf{d)} Sinal exponencial real discreto $x_{1}[n] = (0,80)^{n}$.
    \item \textbf{e)} Sinal senoidal discreto: $x_{2}[n] = 2\cos\left[2\pi(0,3)n + \frac{\pi}{3}\right]$.
    \item \textbf{f)} Sinal exponencial complexo: $x_{3}[n] = (0,9e^{j\pi/10})^{n}$. Para este sinal, deve-se plotar a parte real, a parte imaginária, a magnitude e a fase.
\end{itemize}

\section{Parte II - Operações com Sinais}
Sejam os sinais $A[n] = \{-1,0, 1, 2, 3, 4, 4, 4, 4\}$ e $B[n] = \{1,1,1,1,0,-1,-1,-1,-1\}$. Foram calculados e plotados os seguintes sinais discretos:
\begin{enumerate}
    \item[a)] $X_1[n] = 2A[n] - B[n]$
    \item[b)] $X_2[n] = A[n] + 3B[n]$
    \item[c)] $X_3[n] = X_2[-n]$
\end{enumerate}

\section{Parte III - Plotagem de Sequências}
Também foi necessário plotar as seguintes sequências:
\begin{enumerate}
    \item[a)] $x[n] = 3\delta(n + 2) - 0,5\delta(n) + 5\delta(n - 1) - 4\delta(n - 5)$
    \item[b)] $y[n] = \delta(n + 1) - 2\delta(n - 1) + 5\delta(n - 4)$
\end{enumerate}

\section{Parte IV - Periodicidade de Sequências}
Gerou-se e plotou-se as sequências:
\begin{enumerate}
    \item[a)] $\cos[0,1n - \pi]$, para $-20 \leq n \leq 20$
    \item[b)] $\cos[0,1n - \pi]$, para $-10 \leq n \leq 20$
\end{enumerate}
Além disso, verificou-se analiticamente se as referidas sequências são periódicas e qual o seu período fundamental.

\textit{Os códigos foram implementados na linguagem Octave e estão apresentados a seguir, seguidos das análises dos gráficos obtidos.}

\section{Resultados da Parte I}
Para gerar os sinais fundamentais discretos solicitados no intervalo $-20 \leq n \leq 40$, implementou-se o seguinte código:

\begin{lstlisting}[language=Matlab, caption={Geração dos Sinais Fundamentais Discretos da Parte I}]
n = -20:40; 
x_a = (n == 0);                        % Impulso
x_b = (n >= 0);                        % Degrau
x_c = n .* (n >= 0);                   % Rampa
x_d = (0.80) .^ n;                     % Exponencial real
x_e = 2 * cos(2 * pi * 0.3 * n + pi/3); % Senoidal
x_f = (0.9 * exp(1j * pi / 10)) .^ n;  % Exponencial complexo

% Componentes
x_f_real = real(x_f);
x_f_imag = imag(x_f);
x_f_mag  = abs(x_f);
x_f_fase = angle(x_f);
\end{lstlisting}

\section{Resultados da Parte III}
A plotagem estruturada com múltiplas funções impulso deslocadas no tempo foi realizada definindo-se uma função anônima no Octave:

\begin{lstlisting}[language=Matlab, caption={Plotagem de Sequências com Impulsos da Parte III}]
% Função anônima para impulso (delta de Kronecker)
imp = @(n) (n == 0);
n_III = -10:10; 

% a) x[n] = 3δ(n + 2) − 0,5δ(n) + 5δ(n − 1) − 4δ(n − 5)

% b) y[n] = δ(n + 1) − 2δ(n − 1) + 5δ(n − 4)
y_III_b = imp(n_III + 1) - 2*imp(n_III - 1) + 5*imp(n_III - 4);
\end{lstlisting}

\subsection{Análise de Periodicidade}
Sabe-se que um sinal discreto na forma $\cos(\Omega_0 n + \phi)$ é periódico se, e somente se, a relação $\frac{\Omega_0}{2\pi}$ for um número racional.

No caso proposto, a frequência angular é $\Omega_0 = 0,1$ rad/amostra. Assim, para encontrarmos o período fundamental $N$:
\begin{equation}
    N = \frac{2\pi m}{\Omega_0} = \frac{2\pi m}{0,1} = 20\pi m
\end{equation}

Como não existe um número inteiro $m$ diferente de zero que torne $N$ também um número inteiro (visto que a frequência está atrelada ao número irracional $\pi$), conclui-se que matematicamente **estas sequências não são periódicas**. No domínio discreto, as amostras não se repetem exatamente nos mesmos valores a cada ciclo.
